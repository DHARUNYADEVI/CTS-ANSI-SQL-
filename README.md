# Event Management System - MySQL Project

##  Project Description
This project is part of the CTS Digital Nurture program.  
It demonstrates an Event Management System using MySQL, including table creation, data insertion, and SQL queries for analysis.

---

##  Database Used
MySQL Workbench

---

##  Project Files

### 1. database_setup.sql
Contains database creation script.

### 2. create_tables.sql
Contains all table creation queries:
- users
- events
- sessions
- registrations
- feedback

### 3. insert_data.sql
Contains sample data insertion for all tables.

### 4. answers.sql
Contains SQL queries such as:
- COUNT of registrations
- JOIN between tables
- GROUP BY analysis
- Average rating per event
- Event-wise statistics

##  How to Run

1. Open MySQL Workbench
2. Run `database_setup.sql`
3. Run `create_tables.sql`
4. Run `insert_data.sql`
5. Run `answers.sql`
6. Verify outputs using SELECT queries
